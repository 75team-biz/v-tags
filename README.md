# v-tags
UI components based on Vue.js 2.0


## npm 安装

    npm install v-tags

### 使用

    import Vue from 'vue'
    import vTags from 'v-tags'
    Vue.use(vTags);


## 在页面中直接引入

    <script src="http://s9.qhres.com/static/c0bb7d3ebe9fad62/v-tags-1.0.0.min.js"></script>
