# v-tags
UI components based on Vue.js 2.0


## npm 安装

    npm install v-tags

### 使用

    import Vue from 'vue'
    import vTags from 'v-tags'
    Vue.use(vTags);


## 在页面中直接引入

    <script src="http://s0.qhres.com/!eca213c3/v-tags.min.0.1.5.js"></script>
